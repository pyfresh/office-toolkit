# PDF 合并工具 使用说明书

## 一、软件简介

PDF 合并工具是一款绿色免安装的 Windows 桌面软件，可将多个 PDF 和图片文件按自定义顺序合并为一个 PDF 文档。

**支持格式：** PDF、JPG、JPEG、PNG、BMP、GIF、TIFF、WEBP

## 二、运行环境

- Windows 10 / 11 64 位
- 无需安装 Python 或其他依赖，双击即用

## 三、启动软件

双击 `PDF合并工具.exe`，弹出软件主界面。

## 五、操作指南

### 1. 添加文件

| 方式 | 操作 |
|------|------|
| **拖拽添加** | 从文件资源管理器选中文件，直接拖入软件窗口释放 |
| **按钮添加** | 点击「选择文件」按钮，在弹出的对话框中选取文件（支持多选） |
| **整个文件夹** | 点击「添加整个文件夹」，选择文件夹后自动导入其中所有支持的文件（按文件名排序） |

> 重复添加同一文件会自动去重，不会重复加入列表。

### 2. 调整顺序

合并后的 PDF 将按照列表中从上到下的顺序排列页面。

| 方式 | 操作 |
|------|------|
| **拖拽排序** | 在列表中按住鼠标左键拖动某个条目到目标位置，拖动过程流畅顺滑 |
| **按钮排序** | 选中条目后点击「⬆ 上移」或「⬇ 下移」逐行移动 |
| **多选删除** | 按住 `Ctrl` 点选多个条目，再点「✕ 移除」批量删除 |

### 3. 合并导出

1. 确认文件列表和顺序无误
2. 点击右下角「合并导出 PDF」
3. 选择保存位置和文件名（默认为 `merged_output.pdf`）
4. 等待合并完成（进度条会显示百分比）
5. 完成后弹出提示，点击「是」可直接打开文件所在文件夹

## 六、常见问题

**Q: 图片合并后模糊怎么办？**
A: 原始图片分辨率越高，合并后的效果越好。建议使用扫描件原图。

**Q: 能否只合并 PDF，不加图片？**
A: 可以，软件自动识别文件类型，PDF 和图片可任意混排。

**Q: 合并后的 PDF 页面顺序和列表不一致？**
A: 合并顺序与列表从上到下严格一致，导出前请确认排列无误。

**Q: 软件闪退怎么办？**
A: 可能是文件损坏导致的，请检查源文件能否正常打开。
