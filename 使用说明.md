# PDF 工具箱 使用说明书

## 一、软件简介

PDF 工具箱是一款绿色免安装的 Windows 桌面软件，提供 PDF 合并与编辑两大功能。

**支持格式：** PDF、JPG、JPEG、PNG、BMP、GIF、TIFF、WEBP

## 二、运行环境

- Windows 10 / 11 64 位
- 无需安装 Python 或其他依赖，双击即用

## 三、启动软件

双击 `PDF工具箱.exe`，弹出软件主界面。顶部导航栏可切换「合并」与「编辑」两个功能模块。

## 四、合并模块

### 界面说明

软件采用现代化扁平设计，蓝白配色风格。

  ┌─────────────────────────────────────────────┐
  │  PDF 工具箱              [合并] [编辑]       │
  ├─────────────────────────────────────────────┤
  │  ┌ - - - - - - - - - - - - - - - - - - - ┐  │
  │  │   拖拽 PDF / 图片文件到此处            │  │
  │  │   或点击下方按钮选择文件               │  │
  │  └ - - - - - - - - - - - - - - - - - - - ┘  │
  │  [选择文件]  [添加文件夹]                    │
  │  ┌─ 文件列表（拖拽条目调整顺序）──────────┐  │
  │  │   英文成绩单.pdf                        │  │
  │  │   中文成绩单.pdf                        │  │
  │  │   成绩说明.pdf                          │  │
  │  │   扫描件.jpg                            │  │
  │  └───────────────────────────────────────┘  │
  │  [上移] [下移] [移除] [清空]     [合并导出PDF] │
  └─────────────────────────────────────────────┘

### 操作指南

#### 1. 添加文件

| 方式 | 操作 |
|------|------|
| **拖拽添加** | 从文件资源管理器选中文件，直接拖入软件窗口释放 |
| **按钮添加** | 点击「选择文件」按钮，在弹出的对话框中选取文件（支持多选） |
| **整个文件夹** | 点击「添加文件夹」，选择文件夹后自动导入其中所有支持的文件 |

> 重复添加同一文件会自动去重，不会重复加入列表。

#### 2. 调整顺序

合并后的 PDF 将按照列表中从上到下的顺序排列页面。

| 方式 | 操作 |
|------|------|
| **拖拽排序** | 在列表中按住鼠标左键拖动某个条目到目标位置 |
| **按钮排序** | 选中条目后点击「上移」或「下移」逐行移动 |
| **多选删除** | 按住 Ctrl 点选多个条目，再点「移除」批量删除 |

#### 3. 合并导出

1. 确认文件列表和顺序无误
2. 点击右下角「合并导出 PDF」
3. 选择保存位置和文件名（默认为 `merged_output.pdf`）
4. 等待合并完成（进度条会显示百分比）
5. 完成后弹出提示，点击「是」可直接打开文件所在文件夹

## 五、编辑模块

### 界面说明

  ┌─────────────────────────────────────────────┐
  │  PDF 工具箱              [合并] [编辑]       │
  ├─────────────────────────────────────────────┤
  │  ┌ - - - - - - - - - - - - - - - - - - - ┐  │
  │  │   拖拽 PDF 文件到此处打开              │  │
  │  │   或点击按钮选择文件      [打开 PDF]   │  │
  │  └ - - - - - - - - - - - - - - - - - - - ┘  │
  │  页面预览（点击选中，右键插入，可多选）      │
  │  ┌──────┐ ┌──────┐ ┌──────┐ ┌──────┐       │
  │  │ 第1页 │ │ 第2页 │ │ 第3页 │ │ 第4页 │       │
  │  │ 预览  │ │ 预览  │ │ 预览  │ │ 预览  │       │
  │  └──────┘ └──────┘ └──────┘ └──────┘       │
  │  [删除选中页] [取消选中]         [导出 PDF]   │
  └─────────────────────────────────────────────┘

### 操作指南

#### 1. 打开 PDF

| 方式 | 操作 |
|------|------|
| **拖拽打开** | 将 PDF 文件拖入窗口释放 |
| **按钮打开** | 点击「打开 PDF」选择文件 |

打开后所有页面以缩略图形式实时预览，每页下方标注页码。

#### 2. 选中页面

| 方式 | 操作 |
|------|------|
| **单击选中** | 单击任意页面缩略图即可选中，再次单击取消选中 |
| **多选** | 依次单击多个页面即可逐个追加选中 |
| **取消全部** | 点击「取消选中」一键清空所有选中 |

> 选中的页面会显示蓝色边框高亮。

#### 3. 删除页面

选中一个或多个页面后，点击「删除选中页」即可移除。

#### 4. 插入文件

| 方式 | 操作 |
|------|------|
| **右键插入** | 在缩略图上右键，选择「在此页前插入」或「在此页后插入」 |
| **按钮插入** | 点击「插入文件」将文件追加到末尾 |
| **拖拽插入** | 打开 PDF 后，拖入文件将自动追加到末尾 |

支持插入的格式：PDF、JPG、JPEG、PNG、BMP、GIF、TIFF、WEBP。插入 PDF 文件时会将源文件的所有页面按顺序插入。

#### 5. 导出

1. 确认页面内容和顺序无误
2. 点击「导出 PDF」
3. 选择保存位置和文件名
4. 等待导出完成

## 六、常见问题

**Q: 图片合并/插入后模糊怎么办？**
A: 原始图片分辨率越高效果越好，建议使用扫描件原图。

**Q: 能否只合并 PDF，不加图片？**
A: 可以，软件自动识别文件类型，PDF 和图片可任意混排。

**Q: 合并顺序与列表不一致？**
A: 合并顺序与列表从上到下严格一致，导出前请确认排列无误。

**Q: 编辑模块中如何快速只选某一页？**
A: 先点「取消选中」清空，再单击目标页面即可。

**Q: 软件闪退怎么办？**
A: 可能是源文件损坏导致，请检查源文件能否正常打开。
