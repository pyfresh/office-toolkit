### 操作指南

#### 1. 添加文件

| 方式 | 操作 |
|------|------|
| **拖拽添加** | 从文件资源管理器选中文件，直接拖入窗口上方虚线区域释放 |
| **按钮添加** | 点击「添加文件」按钮，在弹出的对话框中选取文件（支持多选） |

> 支持格式：PDF、JPG、JPEG、PNG、BMP、GIF、TIFF、WEBP。可混合添加 PDF 和图片，添加 PDF 时会导入该 PDF 的全部页面。文件会追加到当前页面列表末尾。

#### 2. 选中页面

| 方式 | 操作 |

| **单击选中** | 单击任意页面缩略图即可选中，再次单击取消选中 |
| **多选** | 依次单击多个页面即可逐个追加选中 |
| **取消全部** | 点击「取消选中」一键清空所有选中 |

|------|------|

| **拖拽追加** | 将文件拖入窗口上方虚线区域，自动追加到末尾 |

> 插入 PDF 文件时会将源文件的所有页面按顺序插入。插入图片时作为单独一页添加。

#### 6. 清空

点击「清空」按钮可一键清除所有页面，恢复初始状态。

#### 7. 导出

1. 确认页面内容和顺序无误
2. 点击右下角「导出 PDF」
3. 选择保存位置和文件名（默认为 `原文件名_编辑.pdf`）
4. 等待导出完成（底部进度条显示百分比）
5. 完成后弹出提示，点击「是」可直接打开文件
